Getting Started with HTML Notes

Notes from, and project for, "Stage 0: Getting Started with HTML" within Udacity's Intro to Programming Nanodegree


World Wide Web Composition

The web is largely composed of HTML documents:
  • HTML = HyperText Markup Language
  • Hypertext = text which contains links to other texts
  • HTML = the language web browsers use to interpret web pages
The components of the web include:
  • The Client = your computer and browser
  • The Internet = the world's largest computer network
  • Servers = computers that host the files that make up the web
The client and servers interact through HTTP:
  • HTTP = HyperText Transfer Protocol
  • Protocol = a standard that specifies how data are transmitted between computers

HTML Components

HTML is made up of:
  • Text content = what you see
  • Markup = what it looks like
  • References to other documents (e.g., images and videos)
HTML Markup is made up of:
  • Elements = components that tell the web browser how to display content on a web page
  • Tags = keywords that tell the web browser how to format the content
HTML Elements:
  • Usually consist of an opening tag, a closing tag, and some content between them


  • General format: <TagName> Contents </TagName>

    • <TagName> = opening tag
    • </TagName> = closing tag

  • Attributes: Additional values accepted by some elements in their opening tags


    • Provide additional information to the browser
    • General Format: <TagName AttributeName = "AttributeValue"> Contents </TagName>


HTML Tags

  • Bold Tag: <b> Contents </b>


  • Em Tag: <em> Contents </em>


  • Anchor Tag: <a href="url"> LinkDisplay </a>

    • Creates a hyperlink to the url & names it LinkDisplay
    • Example of a tag that uses an attribute (href)

  • Image Tag: <img src="url" alt="text">

    • Displays the image hosted at the src attribute value "url" inline with text
    • "url" = path to image, can be a local file on computer
    • "text" = required attribute, displayed for a broken request

  • Break Tag: <br>

    • Creates a line break
    • Inline element
    • Void tag

  • Void Tag = tag with only an opening tag and no end tag because it does not have any content


  • Paragraph Tag: <p> Content </p>

    • Defines a paragraph
    • Is a block element

  • Containers: use to add styles to text

    • <span> Text </span> = Creates an inline container element
    • <div> Text </div> = Creates a block container element

  • Inline and Block:


    • Inline = element continues on current line
    • Block = element starts a new line and makes an invisible box with adjustable height and width

Complete HTML Document Structure

<!DOCTYPE html>
  • Must be the very first thing in HTML document
  • Defines the type of the document so the browser will render it correctly
<html>
  • Opening HTML tag
  • Everything that shows on the webpage is placed within this tag
<head>
  • Opening header tag, defines metadata for webpage
  • Indented to a new level because the <html> tag was opened but not closed
<title> Title </title>
  • Document title
  • Indented another level further because the <head> tag was opened but not closed
</head> Closing header tag
<body>
  • Opening body tag
  • Actual contents of document
</body> Closing body tag
</html> Closing HTML tag
